class pydidas.gui.frames.PyfaiCalibFrame(**kwargs: dict)#

A pyFAI Calibration frame similar to the “pyfai-calib2”, adapted to be used within pydidas.

Note: Because this code is adapted from pyFAI, the names, nomenclature and code structure are different from the rest of pydidas.

Acknowledgements go to the creators of pyFAI for making it freely available.

Widgets must support generic args and kwargs arguments. This means that generic PyQt widgets cannot be created using this method. They can be added, however, using the add_any_widget method.

  • widget_class (type) – The class type of the widget.
